\section*{Abstract}
This project investigates Tube-Based Stochastic Model Predictive Control (SMPC) as a less conservative alternative to Robust Tube MPC for systems subject to stochastic disturbances and uncertainties. The team implements both controllers on a two-mass spring-damper system and introduce the SARAH-M variance-reduced stochastic gradient algorithm as the optimizer for the scenario-based MPC optimization problem. Constraint tightening for the stochastic method is handled probabilistically via Monte Carlo simulation of the closed-loop error dynamics rather than worst-case bounding like the robust method. Simulation results show that the stochastic tube is significantly tighter than the robust worst-case tube, at the cost of increased per-step computation time. We conclude that SARAH-M based SMPC is feasible for moderate real-time requirements and offers a practical trade-off between conservatism and computational complexity.

\section{Introduction}
Control systems are increasingly moving towards complex and uncertain environments.
Model Predictive Control (MPC) is the standard for constrained systems, but its standard deterministic form lacks a formal mechanism to handle perturbations.
Tube-Based Stochastic MPC (SMPC) represents a sophisticated evolution, providing a framework for maintaining stability within a defined "tube" of trajectories.

\subsection{Tube-Based MPC}
In tube-based MPC, the system state is decomposed into two distinct components: the nominal system, which is the controlled, deterministic model that ignores noise,
and the error system, which measures the deviation between the plant and the nominal model, caused by stochastic disturbances (perturbations).
By centering an invariant set around the nominal trajectory, the controller ensures that as long as the nominal state is controlled effectively, the true state of the system is guaranteed to remain within the tube boundaries with a certain probability.

\subsection{Robust vs. Stochastic Tubes}
Traditional Robust Tube MPC assumes worst-case bounded disturbances, which often results in overly conservative control actions.\cite{dey2026adaptivetubempccommon}
Stochastic Tube MPC relaxes this in three ways:
first, by characterizing the probability distribution of the disturbances;
second, by shrinking the constraints based on a risk budget rather than the maximum possible perturbation;
and thirdly, by allowing the system to operate closer to its physical limits, improving the performance without sacrificing the safety margin.

\subsection{Objective}
This project seeks to determine the feasibility of Tube-Based SMPC in practical applications.
First, the authors will outline the application of a Robust Tube-based MPC for a simple spring-damper system.
Next, the authors will implement a Stochastic Tube-based MPC using the SARAH-M algorithm \cite{YANG2024122295}. The SARAH-M algorithm is a variance-reduced stochastic gradient method, and in this project, it is used to solve a scenario-average cost at each time step. The contraint tightening margin for the stochastic method is derived from a Monte Carlo simulation of the closed-loop error dynamics rather than a worst-case bound margin that is used for the robust tube-based MPC. Finally, the two approaches are compared on trajectory tracking performance and per-step computation time.
Finally, the authors will compare the performance profile of the tube-based SMPC to a Robust Tube MPC.

\subsection{State-Space Equation}
\begin{figure}
    \centering    \includegraphics[width=\textwidth]{spring_damper_system.png}
    \label{spring-damper-system}
    \caption{Spring-damper system model.}
\end{figure}

Following the work of Haber\cite{haber2023mpc1}, the physical model of a two-spring, two-damper system is given by Fig.~\ref{spring-damper-system}.
Applying Newton's second law to each mass yields the equations of motion:
$$m_1 \ddot{x}_1 = -(k_1+k_2)x_1 - (d_1+d_2)\dot{x}_1 + k_2 x_2 + d_2 \dot{x}_2$$
$$m_2 \ddot{x}_2 = k_2 x_1 + d_2 \dot{x}_1 - k_2 x_2 - d_2 \dot{x}_2 + F$$

Choose state vector $\mathbf{x} = [x_1, \dot{x}_1, x_2, \dot{x}_2]^\top$, control input $u = F$, and output $y = x_1$.
Then the state-space equation for this system is given by:
$$\dot{\mathbf{x}} = A_c \mathbf{x} + B_c u, \qquad y = C \mathbf{x}$$
here the matrices $A_c$, $B_c$, and $C$ are populated directly from the physical parameters.

The system is then discretized using the forward Euler approximation:
$$A = (I - T_s A_c)^{-1}, \qquad B = A \cdot T_s B_c$$

\subsection{Model Predictive Control}
Model predictive control is an optimization-based control strategy that explicitly uses a dynamic model to predict future plant behavior over a finite prediction horizon $N_p$.
MPC chooses control inputs that minimize a cost function subject to constraints.
At each sampling instant $k$, MPC solves the Optimal Control Problem:

$$\min_{\{u_0, \ldots, u_{N_c-1}\}} \sum_{i=0}^{N_p - 1} \left\| y_{k+i|k} - r_{k+i} \right\|^2_{W_4} + \left\| \Delta u \right\|^2_{W_3}$$
where $y_{k+i|k}$ is the predicted output $i$ steps ahead, $r_{k+i}$ is the reference trajectory, $W_4$ penalizes tracking error over the prediction horizon $N_p$, $W_3$ penalizes control input increments over the control horizon $N_c$, and $\Delta u$ denotes differences between successive control actions.

For a linear system, the predicted output trajectory over the full prediction horizon can be written compactly as:

$$\mathbf{Y} = \mathcal{O} \mathbf{x}_k + \mathcal{M} \mathbf{U}$$

where $\mathbf{Y} \in \mathbb{R}^{N_p r}$ stacks all predicted outputs, $\mathbf{U} \in \mathbb{R}^{N_c m}$ stacks all future inputs, $\mathcal{O}$ is the observability-like matrix encoding free response contributions from the current state, and $\mathcal{M}$ is the lower-triangular Toeplitz matrix encoding input-to-output impulse responses.
Substituting this into the cost function and differentiating yields the unconstrained closed-form optimal gain:

$$\mathbf{U}^* = \underbrace{({\mathcal{M}}^\top W_4 \mathcal{M} + W_3)^{-1} \mathcal{M}^\top W_4}_{\text{gain matrix}} \left( \mathbf{R} - \mathcal{O}\mathbf{x}_k \right)$$

This gain matrix is computed once offline, making the nominal MPC step a simple matrix-vector multiplication.

\subsection{SARAH-M}
SARAH-M is a stochastic optimization method that utilizes Stochastic Recursive Gradient Algorithm (SARAH) with a momentum term to accelerate the optimization process.
The algorithm utilizes an inner and outer loop structure to update a stochastic gradient estimate recursively.\cite{YANG2024122295}
The defining feature is the Heavy-Ball Momentum term in the inner loop, which accelerates the convergence of the algorithm, giving it linear convergence rates for strongly convex functions.

\section{Conclusion}
SMPC can be a viable alternative to Robust MPC when safety considerations do not require an aggressive treatment of the worst case scenario.
While Robust MPC provides absolute saftey guarantees, it does so at the expense of the performance of the system.
The core contribution of this study is the empirical validation that higher risk tolerance can be directly translated to operational gain.
In the simulated scenarios, the SMPC controller achieved a 91.9\% gain in the tube width over the robust controller, primarily because the SMPC can avoid low-probability outlier disturbance events that the robust controller is forced to mitigate.
However, the convergence time of the SARAH-M algorithm is 45.4\% slower, which is a noteworthy time penalty for systems that require fast convergence times on the order of milliseconds or faster.
